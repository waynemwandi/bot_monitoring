## Setup Commands

- Create python virtual environment

```python
python -m venv venv
```

- Activate python environment

```bash
source venv/Scripts/activate
```

- Freeze requirements

```python
pip freeze > requirements.txt
```

- Install requirements
```python
pip install -r requirements.txt
```

- Run FastAPI App

```bash
uvicorn main:app --reload
```
